# NBA Player Performance Forecasting
Master’s dissertation project on forecasting future NBA player performance using deep learning and historical player-season statistics

## What this project is about
This repository contains my Master’s dissertation project on forecasting future NBA player performance using historical player statistics and deep learning.

The repository brings together:
- the dissertation report,
- the modelling notebook,
- the source datasets,
- and a glossary of terms.

## Why this matters
Projecting future player performance is a central problem in basketball analytics.

Teams, analysts, and researchers often want to understand not only how a player performed in the past, but how their performance is likely to develop in future seasons.

This project studies that problem using player-season statistics and neural-network-based regression models.

## What the repository contains
- `Dissertation.pdf`  
  Full dissertation report for the project.

- `NBA_future_performance_submission.ipynb`  
  Main notebook containing the data preparation, modelling, and evaluation workflow.

- `Glossary.xlsx`  
  Supporting glossary for variables and basketball-related terms.

- `Players.csv`, `Seasons_Stats.csv`, `player_data.csv`  
  Data files used to construct the modelling dataset.

## What the project does
The project:
1. starts from historical NBA player-season data,
2. compares the available input structure with a previous study,
3. builds a richer player-performance dataset,
4. expands categorical information into a high-dimensional modelling matrix,
5. trains neural-network models to predict future player-performance measures,
6. evaluates predictive performance using regression metrics.

## What makes this repo stronger than a typical notebook project
This repository is not just code. It includes the dissertation itself, which makes it easier to understand:
- the research question,
- the modelling rationale,
- the evaluation design,
- and the interpretation of results.
